Prosperna Test

Requirements
* Laravel 8.75
* PHP 7.4
* Xampp or Laragon

Installation

* git clone https://github.com/isaacdarcilla/inertia-shop
* composer install
* yarn install
* yarn run watch
* php artisan key:generate
* php artisan migrate --seed
* php artisan serve
